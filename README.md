# Sensor-MPX4115
Sensor MPX 4115 untuk mendeteksi tekanan pada injector di simulasikan dengan proteus
Jika 40 kPa maka injector hidup selama 3 detik

Jika 90 kPa maka injector hidup selama 7,5 detik
